# Aether Auth

A secure authentication web app built with Supabase — supports email/password login, phone OTP verification, and password reset.

## Tech Stack
- HTML, CSS, JavaScript (vanilla)
- Supabase (Auth backend)
- Twilio (Phone OTP via Supabase)

## Features
- Email/password sign up & login
- Phone number login with OTP verification
- Forgot password / reset password flow
- Session persistence
- Responsive glassmorphic UI
